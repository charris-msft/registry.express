About MCP Registry Express

A simplified MCP Registry for private and self-hosted scenarios.

Features

  • Store MCP servers as simple JSON files
  • Import servers from the official MCP Registry
  • Host as a static site (GitHub Pages, Azure Static Web Apps, etc.)
  • API compatible with official MCP Registry
  • No database required

API Endpoints

Endpoint Description
GET /api/v0.1/servers List all servers
GET /api/v0.1/servers/{name}/versions List versions for a server
GET /api/v0.1/servers/{name}/versions/{version} Get specific version details
GET /api/v0.1/servers/{name}/versions/latest Get latest version details

CLI Usage

# Search for servers
mcp-registry search "filesystem"

# Import a server
mcp-registry import "io.github.user/server-name"

# Import all versions
mcp-registry import "io.github.user/server-name" --all-versions

# List local servers
mcp-registry list

# Build the registry
mcp-registry build
